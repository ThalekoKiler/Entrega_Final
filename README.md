# Entrega_Final
Repositório sobre a apresentação final do projeto de Modelagem e Desenvolvimento de Sistemas 

COMO USAR: 

- Ter o MYSQL Workbench instalado em sua máquina
- Ter o Dart instalado em sua máquina
- Baixar e utilizar os Scripts Querry dentro do MYSQL para gerar o banco de dados e os inserts de usuário, sala, etc...
- Baixar e utilizar o código dentro do VisualStudioCode {
  . LEMBRE-SE DE DEIXAR O MYSQL SEMPRE ABERTO;
  . MUDE A SENHA NO CÓDIGO PARA A QUE VOCÊ UTILIZA NO MYSQL;
}
